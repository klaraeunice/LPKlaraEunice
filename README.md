# 💻 Linguagem de Programação - UFPB

Repositório para armazenar as atividades e projetos desenvolvidos durante a disciplina de Linguagem de Programação.

## 🛠️ Tecnologias e Versões
* **Java:** 20 / 25
* **IDE:** IntelliJ IDEA
* **Paradigma:** Programação Orientada a Objetos (POO)

## 📁 Organização do Repositório

O projeto está organizado em módulos independentes:

* **AtividadeBlocoDeNotas**: Sistema de gestão de anotações com tratamento de exceções personalizadas e `ArrayList`.
* **SimuladoProva3**: Implementação de sistema de Cesta de Compras.
* **diarioDeFerias**: Registo de atividades diárias.
*  **ObjetosVoadores**: Registo de diferentes objetos voadores através do uso de Interface.


## 🚀 Como Executar
1. Abra o projeto no IntelliJ.
2. Cada módulo possui a sua própria pasta `src`.
3. Navegue até a classe `Main` dentro do pacote correspondente e execute.
